# AI-Based Student Risk Prediction System

## Problem Statement
Educational institutions struggle to identify students at risk of academic failure early.

## Solution
This system analyzes attendance and internal marks to classify students into risk categories and suggest early interventions.

## Tech Stack
- Python
- Streamlit

## Features
- Risk prediction
- Actionable recommendations
- Simple and intuitive UI

## Deployment
Deployed using Streamlit Community Cloud.

## Future Scope
- ML-based prediction using historical data
- Integration with college databases
- Faculty dashboards
